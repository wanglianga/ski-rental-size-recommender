# 雪具租赁尺码推荐系统

## 原始需求

> 请制作雪具租赁尺码推荐页面，Vue3 页面围绕游客身高、体重、脚长、滑行经验、雪道偏好、取还时间、护具搭配和柜台库存组织。游客在手机上填写信息后得到雪板、雪鞋、头盔、护目镜和护具建议，柜台平板能看到推荐依据、可替换尺码、押金金额、取件号码和换码窗口。页面需要有清楚的步骤感，雪场现场手冷、戴手套、排队催促时也能顺利完成选择，并把不合脚、库存紧张、夜场归还和儿童尺码这些麻烦提前暴露出来。

## 项目简介

面向雪场游客和柜台工作人员的双端雪具租赁尺码推荐系统。游客在手机端按步骤填写身体数据和偏好，系统自动推荐雪板、雪鞋、头盔、护目镜、护具的尺码并提示风险；柜台平板端可查看推荐依据、库存状态、押金金额和取件管理。设计充分考虑雪场现场环境：手冷、戴手套、排队催促时也能顺利完成操作。

## 技术栈

- Vue 3 + TypeScript + Vite 5
- TailwindCSS 3
- Vue Router 4
- lucide-vue-next (图标)
- Composables 状态管理（无需额外库）
- 纯前端应用，数据存储在内存中

## 启动方式

### 前置要求

- Node.js >= 18
- npm >= 9

### 启动步骤

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动服务

```bash
npm run dev
```

访问地址：http://localhost:5173

### Docker 启动方式（推荐）

#### 前置要求

- Docker >= 20
- Docker Compose >= 2

#### 构建并启动

```bash
docker compose up --build
```

后台运行：

```bash
docker compose up --build -d
```

访问地址：http://localhost:3000

#### 停止服务

```bash
docker compose down
```

## 目录结构

```
src/
  types/            类型定义
  data/             尺码对照表和库存模拟数据
  composables/      推荐算法与状态管理
  components/
    guest/          游客端组件（步骤表单、装备推荐卡片、风险提示）
    SnowBackground  雪花粒子动画背景
  pages/            页面组件（首页、游客端、柜台端）
  router/           路由配置
  lib/              工具函数
```

## 主要功能

- **游客手机端**：3 步表单（基本信息 → 滑行经验 → 时间护具）→ 推荐结果 → 确认取件
- **柜台平板端**：订单管理、推荐依据查看、库存替换、取件/归还操作、换码窗口倒计时
- **风险提示**：不合脚警告、库存紧张预警、夜场归还提醒、儿童尺码注意
- **寒冷场景优化**：大触控区域、滑块控件、高对比度配色、步骤感强减少认知负荷
